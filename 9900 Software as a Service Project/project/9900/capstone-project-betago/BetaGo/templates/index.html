About Us

Course & Program Information Search

UNSW students are able to converse with the chatbot to obtain information regarding courses and programs.

Recommender System, Search and Weather Tool

A content based recommender system is utilised in the chatbot system to allow course suggestions for the student based on a preferred subject. The system further functions as a search engine and weather tool should the user desire.

Knowledge Based Query

The chatbot is equipped to understand the underlying meaning of the questions which makes it an excellent teaching assistant. An application of this is demonstrated through the Machine Learning course where students may ask questions regarding content of the course and tutors may add knowledge based content into the chatbots database.
